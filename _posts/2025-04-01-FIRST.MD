# 250-Module Web Development Curriculum for Remix/Next.js Applications

This curriculum for autodidacts and self-starting volunteers is designed to be adaptable to different learning speeds and backgrounds while ensuring all participants ultimately gain the comprehensive knowledge needed to build sophisticated Remix/Next.js applications with various datastores and deployment environments.

### How to Use This Curriculum
- FORK [this repository](https://github.com/RockRapids/onboarding)! EVERYBODY who uses [this repository](https://github.com/RockRapids/onboarding) has to FORK [this repository](https://github.com/RockRapids/onboarding) with the intention of improving [this repository](https://github.com/RockRapids/onboarding).
- Learners can skip modules covering topics they already know; they can skip around, nobody has to do it the right way
- EVERYBODY who uses the curriculum has to understand that each module is intended to build on previous knowledge
- Suggestions for using more AI-assisted recommendations for your learning/dogfooding are included throughout, but you are, of course, not only free to, but encouraged to use AI-assistance where it is not even recommended, ie FORK this repo; make it better.

**KEY PRINCIPLE:  Practical projects and forking the repository to dogfood a better lesson than one that is given reinforce theoretical concepts.Nothing trains you better than training other people how to do what you think you have mastered.** In other words, "*Homines dum docent discunt!*" Be improved by those whom you can improve. Unlike machines, humans learn best by teaching different humans and teach best by learning from different masters.

## Curriculum Overview

This curriculum was dogfooded with the use of AI. Each module represents approximately 5-6 hours of dedicated study and hands-on experimentation.

It is designed to be improved, ie fork this repository, load it into a Claude project knowledge store and then use your progression through the modules to make those modules better [with the help of AI].

The FULL training roadmap is designed to take enthusiastic, autodidactic volunteers from whereever they are in the learning curve ... from needind to acquire very basic computer literacy TO developing sophisticated Remix/Next.js applications that work with various datastores and deployment environments. 

With the use of AI assistance, autodidactic learners can also take all kinds of side paths ... BUT TO WORK ON THE ROCK RAPIDS COMMUNITY APP ECOSYSTEM, all volunteers are required to be "familiar" with this roadmap.  "Familiar" means that you have forked this repository and have improved it significantly.

- Maintain a Q&A forum for each module

- Develop a resource library for supplementary materials

### AI Assistance Framework

- Teach verification of AI-provided information

- Module completion projects
- Knowledge check quizzes
- Peer review sessions
- Portfolio building throughout curriculum

### Instructor Support
- Provide instructor guides for complex topics
- Create troubleshooting guides for common issues
- Develop answer keys for exercises
- Include discussion prompts for group learning

### Adaptive Learning Paths
- Mark prerequisite relationships between modules
- Create "fast track" paths for experienced developers
- Develop "deep dive" optional materials for each module
- Maintain a knowledge base of common questions and challenges
